ChatList

Возможности

Мульти-модель

Один промт — параллельные запросы в GPT, Claude, Gemini и другие модели через OpenRouter.

Сравнение ответов

Результаты в таблице: отметьте лучшие, сохраните в базу или экспортируйте в Markdown / JSON.

AI-ассистент

Встроенное улучшение промтов перед отправкой — с выбором цели: код, анализ, креатив.

Локальные данные

Промты, модели и результаты в SQLite. API-ключи только в файле .env на вашем ПК.

Быстрый старт

  1. Скачайте и установите ChatList (или запустите portable-версию).
  2. Скопируйте .env.example в .env и укажите OPENROUTER_API_KEY.
  3. Введите промт, нажмите Отправить — ответы появятся в таблице.
  4. Отметьте нужные строки и нажмите Сохранить.

Для разработчиков

Исходный код открыт (MIT). Запуск из репозитория:

python -m venv .venv → pip install -r requirements.txt → python main.py
