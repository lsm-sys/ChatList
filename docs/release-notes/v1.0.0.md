# ChatList v1.0.0

Первый публичный релиз — приложение для отправки одного промта в несколько нейросетей и сравнения ответов.

## Что нового

- Главное окно: ввод промта, отправка в активные модели, сравнение ответов
- Управление моделями OpenRouter и другими OpenAI-совместимыми API
- Сохранение промтов и результатов в SQLite
- AI-ассистент для улучшения промтов
- Экспорт выбранных результатов в Markdown и JSON
- Светлая и тёмная тема
- Установщик Windows (Inno Setup) с поддержкой удаления

## Установка (Windows)

1. Скачайте **ChatList-Setup-1.0.0.exe** (рекомендуется) или **ChatList-1.0.0.exe** (portable).
2. Запустите установщик и следуйте шагам мастера.
3. В папке программы скопируйте `.env.example` в `.env` и укажите ключ:

```
OPENROUTER_API_KEY=sk-or-v1-ваш-ключ
```

## Системные требования

- Windows 10/11 (64-bit)
- API-ключ [OpenRouter](https://openrouter.ai/)
