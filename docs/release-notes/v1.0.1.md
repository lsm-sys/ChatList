# ChatList v1.0.1

Релиз инфраструктуры публикации: лендинг, документация и автоматизация выпусков.

## Что нового

- HTML-лендинг для GitHub Pages (`docs/index.html`)
- Пошаговая инструкция публикации (`docs/PUBLISHING.md`)
- Скрипт `publish-release.ps1` — сборка, тег, push и GitHub Release в одном шаге
- GitHub Actions workflow для автодеплоя Pages
- Шаблоны заметок к релизам (`docs/release-notes/`)

## Исправления

- `publish-release.ps1` автоматически отправляет тег на GitHub
- Удалён ошибочный пустой Release v1.0.1

## Установка (Windows)

1. Скачайте **ChatList-Setup-1.0.1.exe** (рекомендуется) или **ChatList-1.0.1.exe** (portable).
2. Запустите установщик и следуйте шагам мастера.
3. В папке программы скопируйте `.env.example` в `.env` и укажите ключ:

```
OPENROUTER_API_KEY=sk-or-v1-ваш-ключ
```

## Системные требования

- Windows 10/11 (64-bit)
- API-ключ [OpenRouter](https://openrouter.ai/)
